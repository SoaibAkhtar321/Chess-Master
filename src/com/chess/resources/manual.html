Chess: User Manual

Before the Game Starts

Before your game starts, you will get some fancy options to adjust the gameplay. Let's go through them from top to bottom.

The Menu

Game menu: If you click on 'game' you get the possibility to quit already now. Do not even think about it.

Help menu: The help menu will allow you to show this pretty page (what a pleasure). Also, it has a page for chess rules in general.

The Color Choice

Now hold your horses: you get to choose the color of your pieces! Wait, WHAT?

Okay, thing is: the white pieces always start. Never mind. If 'white' is chosen, the white pieces will start at the bottom and play upwards, the black pieces start from top and play to the bottom. If 'black' is chosen, vice versa.

The Rule Set

I know, where 'you come from', you 'don't do it like that'. Okay. Let's make you happy, you can play like 'people' play where you come from.

  • Touched/Moved: If this is on, you will be forced to move a chess piece if you touched it before. No 'ah no, let's try another maybe'.
  • En Passant: Special pawn hit - better check rules on this one.
  • Timeout: This will end the game if a board situation happens about three times or 50 irrelevant moves are made.

The Game Mode

This option lets you play with the AI. If you like, you can also play with a friend on the same computer (of course you have friends). Or you can challenge both AI bots (and maybe learn something).

The Difficulty

We want to make you happy, so you get the possibility to lower the difficulty. But don't go bragging afterward.

The Loading

If you would like to load an existing game, use this nice little button here. Clicking it will open a dialog, where you can either enter a FEN or PGN code or upload a PGN file. Nice.
